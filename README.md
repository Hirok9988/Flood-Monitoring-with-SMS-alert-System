# IoT Flood Monitoring & SMS Alert System

This repository contains the source code and project files for an IoT-based flood monitoring system using ESP8266 and GSM SIM900A.

## Folder Structure
- code/ → Arduino source code
- images/ → Project setup images

## Hardware Used
- ESP8266
- Ultrasonic Sensor
- Flood Switch Sensor
- GSM SIM900A
- 16×2 LCD
- Bread Board
- Jumper wires
